About the Project

Geode is a unique collection of Geode NFT's, comprised of various backgrounds, generations, and modifiers.

Utility

  1. Holders will be invited to an exclusive Discord group where we can share ideas, upcoming projects, and create and awesome social environment.
  2. Holders will be entitled to a percentage of all sales meaning the percentage of every sale will be held in a wallet and distrubuted to holders at the end of each month.
  3. More to come!!
Phase 1 will be finishing the art and generating each NFT. I wrote a program that takes layers of images and combines them to create NFTs. This program will be available in its basic form on my github after mint. The version used to generate Geodes will be a modified version of this program.
Phase 2 will include generation combination for rarer NFT's. This tool will allow holders to combine multiple Geode generations into 1 higher rarer generations. I am also thinking about combinations for gen 5 which is the highest generation level.
Phase 3 will essentially be mutating Geodes like BAYC mutates their apes using an NFT which will be created shortly after generation combination. These mutations will modify the overall look of each background and the Geode itself. More info will be available on mutations after mint.
I am no artist, so that is my only current holding point.
